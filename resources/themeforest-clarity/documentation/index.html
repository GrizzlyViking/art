Project Structure

All the directories and files are well organized. Project tree is shown below.

  • assets
    It contains the basic template files
    • components
      Components with its internal structure
      • bootstrap
      • elegant-icons
      • et-line-icons
      • font-awesome
      • magnific-popup
      • owl-carousel
      • themify-icons
    • css
      Single CSS files
    • fonts
      Additional fonts
    • img
      Basic template images
    • js
      Single JS files
    • scss
      Template SASS-Files Remove this from release
      • external
      • includes
      • layout
      • mixins
      • theme
      • ui
      • theme.scss
        Main scss theme file (contains the inclusion of all files)
  • images
    Page images
  • favicon.ico
  • index.html
